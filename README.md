# imessage-search

```
usage: im-search.py [-h] [-d path] [-s query] [-c number]

This tool searches iMessages chat.db with a given search query and/or contact
number

optional arguments:
  -h, --help  show this help message and exit
  -d path     Set path to iMessage chat.db (Defaults to current console user
              chat.db)
  -s query    Search query
  -c number   Contact number
```
